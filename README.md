# Fun Club Impact Analytics

## Project Overview

Fun Club Impact Analytics is a data-driven project created to measure student participation, volunteer engagement, instructional sessions, and program growth for Fun Club, a student-led educational initiative supporting Azerbaijani students through English learning, technology skills, mentorship, and college preparation.

This project demonstrates how data analytics can be used to measure community impact and tell a clearer story about educational outreach.

## About Fun Club

Fun Club is a student-led community initiative focused on helping students gain access to academic support, English practice, basic technology skills, mentorship, and college preparation resources.

## Project Highlights

- Tracked participation across four program areas: English Learning, Technology Skills, College Prep, and Mentorship.
- Created visualizations to show student engagement and volunteer contribution over time.
- Built a simple impact-measurement framework that can be used by student-led organizations and nonprofits.
- Connected community service with data analytics and program evaluation.
